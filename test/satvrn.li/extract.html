✦ How To Extract ​

Windows ​

Install 7-Zip or alternative software. For windows users, we recommend 7-Zip as it is both free and open source. Alternatives include NanaZip and WinRAR.

Click here for a video tutorial. ​

  1. Right click the file you wish to open
  2. Select 7-ZIP — under “show more options” on windows 11.
  3. Extract here > Enter password > OK (password: star)
  4. Proceed with your new, unzipped folder.

MacOS ​

Install Keka or alternative software. For Mac users, we recommend Keka as it is both free and open source. Alternatives include The Unarchiver and PeaZip.

Note: Only the AppStore version is paid, while the download on their site is free.

Click here for a video tutorial. ​

  1. Right click the file you wish to open
  2. Select open with > choose "Keka"
  3. Select location if asked > Enter password > Done (password: star)
  4. Proceed with your new, unzipped folder.

Archive utility can open encryptyed files in .zip format, but not .7z. Files will vary.
